553 class Pattern(object):
554  """
555  A pattern is a string, possibly with a % substitution character. From the GNU make manual:
556 
557  '%' characters in pattern rules can be quoted with precending backslashes ('\'). Backslashes that
558  would otherwise quote '%' charcters can be quoted with more backslashes. Backslashes that
559  quote '%' characters or other backslashes are removed from the pattern before it is compared t
560  file names or has a stem substituted into it. Backslashes that are not in danger of quoting '%'
561  characters go unmolested. For example, the pattern the\%weird\\%pattern\\ has `the%weird\' preceding
562  the operative '%' character, and 'pattern\\' following it. The final two backslashes are left alone
563  because they cannot affect any '%' character.
564 
565  This insane behavior probably doesn't matter, but we're compatible just for shits and giggles.
566  """
